　定数$a>0$に対し，曲線$y=a\tan x$の$\displaystyle 0 \leqq x < \frac{\pi}{2}$の部分を$C_1$，
曲線$y=\sin2x$の$\displaystyle 0 \leqq x < \frac{\pi}{2}$の部分を$C_2$とする。
以下の問いに答えよ。
\begin{description}
\item[(1)]$C_1$と$C_2$が原点以外に交点をもつための$a$の条件を求めよ。
\item[(2)]$a$が(1)の条件を満たすとき，原点以外の$C_1$と$C_2$の交点を$P$とし，$P$の$x$座標を$p$とする。
$P$における$C_1$と$C_2$のそれぞれの接線が直交するとき，$a$および$\cos2p$の値を求めよ。
\item[(3)]$a$が(2)で求めた値のとき，$C_1$と$C_2$で囲まれた図形の面積を求めよ。
\end{description}
\end{flushleft}

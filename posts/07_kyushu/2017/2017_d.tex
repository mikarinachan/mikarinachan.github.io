{\huge 4}
　$k$を$k\geqq0$，$k\neq1$を満たす実数として，関数$f(x)$を
\[ f(x)={ \{ 1-(1-k)x \} }^\frac{1}{1-k} \]
で定める。ただし，関数$f(x)$の定義域は，
$0 \leqq k < 1$のとき$\displaystyle 0 \leqq x \leqq \frac{1}{1-k}$であり，
$k>1$のとき$x\geqq0$である。以下の問いに答えよ。
\begin{description}
\item[(1)]導関数$f'(x)$を求めよ。
\item[(2)]$k=0$のとき，$0<k<1$のとき，$1<k$のときのそれぞれの場合について，
関数$y=f(x)$の増減，グラフの凹凸，座標軸との交点を調べてグラフをかけ。
\item[(3)]$x\geqq0$であるとき，
\[ \lim_{k\to1+0} { \{ 1-(1-k)x \} }^\frac{1}{1-k} \]
を求めよ。ここで，自然対数の底$e$が
$\displaystyle e=\lim_{h\to0}{ (1+h) }^\frac{1}{h}$を満たすことを用いてよい。
\end{description}
\end{flushleft}

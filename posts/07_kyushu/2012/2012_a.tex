{\huge 1}
　以下の問いに答えよ。
\begin{description}
\item[(1)]$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$のとき，
$\sin x-x\leqq0$が成り立つことを示せ。
\item[(2)]$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$のとき，
$\sin x-x+ax^3\geqq0$が成り立つような正の数$a$を1つ定めよ。
\item[(3)]$\displaystyle 0 < |x| \leqq \frac{\pi}{2}$のとき，
\[ \left| \frac{\sin x}{x}-1 \right| \leqq ax^2 \]
が成り立つことを示せ。ただし，$a$は(2)で定めた値とする。
\item[(4)]曲線$y=\sin x$と直線$y=x$および$\displaystyle x=\frac{\pi}{2}$で囲まれた図形を
$x$軸のまわりに1回転してできる回転体の体積を求めよ。
\end{description}
\end{flushleft}

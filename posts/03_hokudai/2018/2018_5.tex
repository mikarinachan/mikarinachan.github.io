{\huge 5}
　2つの関数
\[ f(x)=\cos x, \hspace{1zw}
g(x)=\sqrt{\frac{\pi^2}{2}-x^2}-\frac{\pi}{2} \]
がある。
\begin{description}
\item[(1)]$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$のとき，
不等式$\displaystyle\frac{2}{\pi}x\leqq\sin x$が成り立つことを示せ。
\item[(2)]$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$のとき，
不等式$g(x) \leqq f(x)$が成り立つことを示せ。
\item[(3)]$\displaystyle 0 \leqq x \leqq \frac{\pi}{2}$の範囲において，
2つの曲線$y=f(x)$，$y=g(x)$および$y$軸が囲む部分の面積を求めよ。
\end{description}
\end{flushleft}
